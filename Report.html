3. Project Development Checklist

Technical Implementation (200 points)

  • 7 Content Pages: The website includes a minimum of 7 unique content pages.
  • DTD Validation: The site validates against XHTML 1.1 STRICT or HTML5.
  • Homepage: Includes a home page with the filename index.html.
  • Responsive Design: The website is responsive for all devices (use Bootstrap, media queries, etc.).
  • Intrasite and Extrasite Links: Has both internal links (within the site) and external links (outside the site).
  • Document Links: At least one page has an anchor link to a section within the same document.
  • Semantic HTML: HTML is structured semantically for accessibility and readability.
  • Form with Validation: Contains at least one form with five elements, validated using JavaScript or HTML5/CSS3.
  • Aesthetics and Navigation: The site is visually appealing with an effective navigation scheme.
  • Graphics: Includes at least one PNG, one JPG, and one GIF image.
  • Efficient Design: The site is optimized for download speed and organized backend structure.
  • Mobile App Packaging: The website is packaged as an Android app using Cordova or WebIntoApp.
  • Device-Sized Layout: Properly optimized for mobile device viewing.

CSS Requirements

  • Responsive CSS: Uses media queries or JavaScript for layout adjustments on different screen sizes.
  • CSS Formatting: All formatting is done using CSS.
  • CSS Layout: All layout is structured using CSS.
  • Styled Links: Links are presented as a styled unordered or definition list.
  • Floated DIV: At least one DIV element is floated.
  • Clear Property: Uses the clear property at least once.
  • Box Model: Uses box model properties (border, margin, padding).
  • Multiple CSS Selectors: Utilizes multiple CSS selectors (IDs, classes, etc.).
  • CSS Properties: Includes background, image, table, and text CSS properties.

JavaScript or HTML5/CSS3 Requirements

  • Rollover Buttons: At least one button set with JavaScript or CSS3 hover effects for rollover functionality.
  • Form Validation: JavaScript or HTML5/CSS3 checks to ensure form fields aren’t left blank.

Hosting Requirements

  • LAMP/LEMP Stack: Hosted on a VPS with a LAMP or LEMP environment.
  • HTTPS: The site uses HTTPS for secure connections.
  • Domain & DNS: Configured with a properly set up domain name and DNS.

Additional Grading Criteria (100 points)

  • Just-In-Time Assignments (25 points): Completion of assignments like server setup, documentation, and pitch video creation.
  • Quality Testing and Validation (25 points): Tested across multiple devices and browsers with issues resolved.
  • Production Documentation (25 points): Includes functional specification, user acceptance tests, mock-ups, flowcharts, and preproduction documentation.
  • Client Management (25 points): Incorporates feedback from TA and classmates, showing improvements in response to guidance
